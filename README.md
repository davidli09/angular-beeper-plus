Angular Drum Machine
--------------------

Just a little experiment with [AngularJS](https://github.com/angular/angular.js "AngularJS"), HTML5 Audio (via [howler.js](https://github.com/goldfire/howler.js "howler.js")), and a simple [Sinatra](https://github.com/sinatra/sinatra "Sinatra") wrapper.

DEMO: [http://drums.dojosto.com](http://drums.dojosto.com)
